Clamp Size Docs

To use size() function in your scss project you need to:

  1. Install the clamp-size package
  2. Import the clamp-size.scss file
  3. Fill in $breakpoints array and $base-rem-size variable
  4. Use the size() function

There are multiple ways you can use the size() function.

Feel free to use inspector tools to see how this page works.

Performance note: consider using css variables to avoid similar calculations.

Basic Usage

font-size: size(18);

This will automatically convert 18px to rem based on the $base-rem-size variable and would return "font-size: 1.125rem" (if the $base-rem-size is set to 16).

I'm always 18!

Fluid sizes

font-size: size(14, 50);

This will make font-size to grow from 14px to 50px from the smallest to the largest breakpoint.

Resize window to see me grow!

Reversed sizes

font-size: size(50, 14);

This will make font-size to shrink from 50px to 14px from the smallest to the largest breakpoint.

I will shrink!

Breakpoints

font-size: size(14, 50, 'sm', 'lg');

This will make font-size to grow from 14px to 50px from the 'sm' breakpoint to the 'lg' breakpoint.

I will grow! Sometimes.

Of cource you can use reversed values here too.

font-size: size(50, 14, 'sm', 'lg');

This will make font-size to shrink from 50px to 14px from the 'sm' breakpoint to the 'lg' breakpoint.

I will shrink! Sometimes.

Advanced Usage

In some cases we would need a different grow speed for different breakpoints.

.advanced {
    font-size: size(14, 18, 'xs', 'md');
    @media (min-width: #{size(768)}) {
        font-size: size(18, 50, 'md', 'xxl');
    }
}

This will make font-size to grow from 14px to 18px from the 'xs' breakpoint to the 'md' breakpoint and shrink from 18px to 50px from the 'md' breakpoint to the 'xxl' breakpoint.

Current Page Setup

For this page I have used the following breakpoints:

  • xs: 320px
  • sm: 576px
  • md: 768px
  • lg: 992px
  • xl: 1200px
  • xxl: 1400px

And this is how I've set it up:

    $breakpoints: (
        xs: size(320),
        sm: size(576),
        md: size(768),
        lg: size(992),
        xl: size(1200),
        xxl: size(1400)
    );
    $base-rem-size: 16;
